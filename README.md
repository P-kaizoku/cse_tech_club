# Instructions

1. This is an individual assignment focused on core front-end skills: HTML5, CSS3,
    JavaScript, DOM, and responsive design.
2. Problem Statement: Design and develop a responsive “Technical Club Portal”
    website for the CSE department.
3. The website must contain at least four pages: Home, Events, Team, and Contact/Join
    Us.
4. Use semantic HTML5 tags and an external CSS file for styling.
5. Implement layout using modern CSS (Flexbox or Grid) and ensure the site works well
    on mobile, tablet, and desktop (basic responsive design).
6. Add a Contact/Registration form with the following fields: name, email, phone,
    year/section, and area of interest.
7. Implement **client** ‑ **side validation** in JavaScript for the form: required fields, email
    pattern, phone length, and basic error messages displayed near the inputs.
8. Add at least one JavaScript‑based interaction such as: show/hide sections, dark/light
    theme toggle, or dynamic event list created from an array.
9. Deliverables:
    ○ ZIP file containing all HTML, CSS, JS, and asset files (well‑organized folder
       structure).
    ○ A 2 ‑page PDF explaining the site structure, key design decisions, and
       screenshots of each page.
10. Placement relevance: The topics in this assignment (HTML/CSS fundamentals,
    responsive design, JavaScript and DOM manipulation, form validation) directly match
    the front‑end rounds of companies such as Google, Microsoft, Amazon, Uber,
